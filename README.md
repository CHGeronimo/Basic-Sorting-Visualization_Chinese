# Basic Sorting Visualization_Chinese
## 基本排序可视化系统 版本1.00,开发时间2021.12.7
 
___
 
## 背景
 本项目旨在利用可视化程序帮助同学们更好理解排序算法的运行过程。（可用作c语言期末大作业）
## 安装
本项目基于c语言编写，您可以直接选择下载msi安装运行，也可以下载源码进行修改，对未打包程序的修改需要Easyx图形库的支持，请您提前安装。
